# Campus IoT Dashboard V6

A React dashboard built with [Material UI](https://mui.com/) based on the official MUI dashboard template.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Structure

- **src/dashboard/** - Main dashboard components (from MUI template)
- **src/shared-theme/** - Theme configuration and customizations
- **D-Dashboard/MUI_REPO/** - Cloned MUI Material UI repository for reference

## Tech Stack

- React 18
- Material UI v7
- MUI X (Charts, Data Grid, Date Pickers, Tree View)
- Vite
- TypeScript
